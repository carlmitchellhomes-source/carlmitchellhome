# Carl Mitchell Home

## Project Description
Carl Mitchell Home is a web application designed to provide users with a seamless experience in managing various real estate properties. The project focuses on user-friendly interfaces, robust functionalities, and a modern technology stack.

## Fixes Applied
- Resolved display issues on mobile view.
- Fixed bugs related to user authentication.
- Various performance enhancements to optimize loading times.

## Features
- User authentication and authorization
- Property listing and filtering
- Interactive maps for property locations
- Favorites and comparison functionalities

## Installation Instructions
To install and run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/carlmitchellhomes-source/carlmitchellhome.git
   ```
2. Navigate to the project directory:
   ```bash
   cd carlmitchellhome
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
